About

Opengov.hk is a project of Open Data Hong Kong to make requesting for information under the Hong Kong government's Code on Access to Information easier, more transparent, and hopefully fairer by clearly showing how the government treats the requests.

Try making a request now, or seeing which requests have already been made! If you want more information about how to make a request, please read this.
